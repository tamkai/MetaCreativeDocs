\[注釈A：MCP(Model Context Protocol)とAI併用の技術的背景\]

MCP(Model Context Protocol)は、AI

がローカル環境のファイルシステムやデータベース、外部APIなどと安全に接続するためのプロトコルである。ClaudeのデスクトップアプリはこのMCPに対応しており、ユーザーは自分のPC上のデータをAIに読ませたり、特定のツールを使わせたりできる。

Opiが「2人いるとガラッと状況が変わる」と表現したのは、技術的には以下のような状態を指している：

  • ChatGPT：一般的な知識とコード生成に強いが、ユーザー固有の環境情報にアクセスできない
  • Claude(MCP接続済み)：ユーザーのローカル環境を「見る」ことができ、具体的なファイル構造やコードを参照しながら提案できる

この「複数AI併用」という発想は、2025年現在、AI活用の最先端実践者に共通する戦略である。単一のAIを「完璧なアシスタント」として扱うのではなく、それぞれのAIの得意領域を理解し、使い分ける。これはまさに、人間が複数の専門家にコンサルティングを依頼するのと同じ思考法だ。

\[注釈B：「できるはずだ感」と自己効力感(Self-Efficacy)の理論的接続\]

タムカイが直感的に「できるはずだ感」と名付けたこの概念は、心理学における自己効力感(Self-Efficacy)ときわめて近い。

自己効力感とは、カナダの心理学者アルバート・バンデューラが提唱した概念で、「ある状況において必要な行動をうまく遂行できると、自分の可能性を認知していること」を指す。簡単に言えば、「自分ならできる」「きっとうまくいく」と思える認知状態である。

バンデューラは、自己効力感を生み出す基礎として以下の4つの要因を挙げている：

  1. 達成経験(Mastery Experience)：自分自身が何かを達成したり、成功したりした経験(最も重要)
  2. 代理経験(Vicarious Experience)：自分以外の他人が何かを達成したり成功したりすることを観察すること
  3. 言語的説得(Verbal Persuasion)：周囲から「君ならできる!」「よくやった!」とポジティブな言葉で励まされたり、褒められたりすること
  4. 情動的喚起(Emotional Arousal)：心身の健康を保ち、穏やかな精神状態でいること。ワクワク・ドキドキといった高揚感を感じる環境

タムカイが「成功体験」と「その質」にこだわったのは、バンデューラの「達成経験」概念と完全に一致している。しかもタムカイは、「課題が出て100点取れた」ような受動的成功体験ではなく、「何もないところから立ち上がるところまでやった」能動的達成体験こそが重要だと指摘する。

これは学術的にも正しい。自己効力感の研究では、自分の力で困難を乗り越えた経験ほど、強い効力感を生むことが実証されている。逆に、他者が設定した課題を指示通りにこなしただけの経験は、効力感をそれほど高めない。

  6. スポンサー獲得：KDDIがスポンサーに、電通DIメディアパートナーズが取りまとめ

2-1. 「グロス」という幻想との戦い

対話は、ビジネス戦略の具体論へと移行する。

Opi

対面なんで生々しい話でいこう。ビジネスとして考えた時に、コンテンツビジネスってどうやったって、お金をいただくかと考えた時に、タムカイさんのプランにも書いてあったけど、法人向けにしてガサを取るか、個人向けにして、ただ個人向けだとガサが取れないから、実際しんどいよねみたいな話があるじゃないですか。

これは、あらゆるB2BとB2Cのビジネスが直面する根本的ジレンマだ。

  • 法人向け(B2B)：一件あたりの単価が高く、「グロス(総売上)」を稼ぎやすい。しかし営業コストも高く、意思決定が遅い
  • 個人向け(B2C)：単価が低く、大量に売らないとグロスが立たない。しかし個々の顧客との関係構築がしやすい

Opiはまず、法人向けはイメージしやすいと言う。

Opi

法人向けはパッとイメージつくじゃないですか。これやって、セミナーやって、そこでワークショップがいくつか何種類かあって、この人にはこのワークショップみたいな風にやって、30万です、40万ですって売ればいいってイメージ湧くじゃないですか。

これは、タムカイが富士通で培ってきた「研修ビジネス」の典型的なモデルである。企業の人事部から依頼を受け、ワークショップを提供し、30〜40万円の対価を得る。この形式なら、年間10〜20社と契約すれば、300〜800万円の売上が立つ。

しかし、Opiはここで個人向けの可能性を提示する。

Opi

個人向けで考えた時で言うと、なんだろうな。ラジオをやってて、すごく実は周辺情報がいっぱい実はあるじゃないですか。別にAIにかければ周辺情報を洗い出せると思うんですよ。なんで、何かしらそのアプリがサブスクのかどうなのかわかんないですけど、何かしらそこに入会すると、知りたい知識が全部リストアップされている状態みたいな。感じで、これ入っておくとメンバーシップみたいな感じで1000円でもいいんだけど、月1000円払っておくと困った時の情報アクセス、すごい整備されているぞみたいな。多分そっちの方なんだろうなって思ったんですよ。

ここでOpiが提案しているのは、知識のデータベース化とサブスクリプションモデルだ。

  • 月額1000円
  • 会員になると、創造性に関する膨大な情報アーカイブにアクセスできる
  • ラジオ、論考、ワークショップ記録、推薦図書リスト etc.

タムカイ

ここには、B2BとB2Cが融合する新しいモデルの萌芽がある。

  • コアは個人向けコミュニティ(月1000円)
  • しかしそのコミュニティに、企業の意思決定者が個人として参加している
  • 結果として、企業案件が「自然発生的に」生まれる

タムカイはさらに続ける。

タムカイ

これは極めて重要な洞察である。

従来の新規事業モデルでは、「大きなグロスを最初から狙う」ことが前提とされる。しかし、その結果何が起こるか。

  • 8割できているプロダクトを、9割に改善することに時間を費やす
  • 完璧を目指すあまり、リリースが遅れる
  • 結果として、顧客の反応を見る前に疲弊する

タムカイはこの罠に気づいている。そして、Opiの提案に触発されて、全く別のアプローチを発見する。

  1. ストーリー(Story)：そのキャラクターが世界の中でどんな旅をするか。どう成長し、どう変容するか

「沼らせる」。この俗語的な表現には、実は深い戦略が込められている。

「沼」とは、オタクカルチャーで使われる言葉で、「ある対象にのめり込んで抜け出せなくなる状態」を指す。例えば「アイドル沼にハマった」「ガンプラ沼から抜け出せない」といった使い方をする。

Opiが「沼らせる」と言ったとき、それは単に「依存させる」という意味ではない。むしろ、知的好奇心が次々に刺激され、学びと発見の連鎖が止まらなくなる状態を指している。

この「沼らせる」戦略は、具体的には以下のような設計になる：

  1. 入り口は小さく：月1000円で入れる。まずは1本のラジオを聞く
  2. キーワードの発見：「できるはずだ感」という面白いキーワードに出会う
  3. 深掘りのリソース：「できるはずだ感」について、関連する論考、推薦図書、過去のセッション記録が100個ある
  4. 新しいキーワードへの連鎖：読んでいくうちに「自己効力感」「達成経験」「代理経験」といった新しいキーワードに出会う
  5. 自分も参加したくなる：「自分もOpiさんやタムカイさんとセッションしたい」と思う
  6. さらなる深み：セッションすると、自分の思考が論考として形になる。その論考を他の会員が読む

このサイクルが回り始めると、コミュニティは自己増殖的に成長していく。
